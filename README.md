
# Final Project

**Final project repository for SAL 603**

## Project Description

Professional Sports League W/L% (NFL, MLB, NHL, NBA, and EPL) Correlation Analysis and Borda Scores for creation of power rankigns

## Technologies Used

- Python

## Project Goals

To identify which offensive and defensive performance metrics are most strongly linked to team win percentage in the NFL, MLB, NHL, NBA, and EPL. We aimed to find out whether team success is more closely related to peak single-season statistical performance or to consistent output over multiple seasons. We also wanted to compare the different leagues to see how results varied across leagues. 

Used Borda Scores to create a power ranking for each league, providing a standardized method for comparing statistical outputs across individual seasons, the last five seasons, and evaluating team consistency in teams’ statistical performance. 


## Research Questions

1. Which offensive and defensive performance metrics are most strongly correlated with team win percentage across professional sports leagues (NFL, NBA, MLB, NHL, EPL)?

2. Is team success more strongly associated with peak single-season statistical output or with consistent performance sustained across multiple seasons?
